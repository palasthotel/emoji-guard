If you ever had problems migrating your data and lost all your emojis then you know why.

== Description ==

Checks if emojis were migrated correctly and if not displays a warning.

== Installation ==

1. Upload `emoji-guard.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `emoji-guard` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
